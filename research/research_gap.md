# Research Gap

## Existing Work

Most cyberbullying detection systems focus only on classification.

Existing models:
- SVM
- Random Forest
- LSTM
- BERT

## Limitations

1. Lack of explainability.
2. Emotion information is ignored.
3. Severity levels are not predicted.
4. Decision-making process is hidden.

## Proposed Solution

This project proposes an Explainable Emotion-Aware Cyberbullying Detection Framework.

Features:

- Cyberbullying Detection
- Sentiment Analysis
- Emotion Analysis
- Severity Prediction
- SHAP Explainability

Expected Outcome:

A transparent and accurate cyberbullying detection system.
